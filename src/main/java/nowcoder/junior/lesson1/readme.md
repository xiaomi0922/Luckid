1、算法时间复杂度
插入、选择、冒泡排序  时间复杂度为O(N)   空间复杂度为O(1)
归并排序  时间复杂度O(NlogN)   空间复杂度O(N)
快速排序  时间复杂度O(NlogN)   空间复杂度O(logN)
堆排序   时间复杂度O(NlogN)   空间复杂度O(1)
桶排序 (计数排序、基数排序)   不是基于比较而是基于数据状况的排序
2、Array.sort中的关键点
(1) 在基础排序时, 数量小于60的时候, 直接进行插入排序, 大于60时, 进行merge排序或者quick排序
(2) 在自己定义比较器排序时, 会使用merger排序, 因为涉及到排序的稳定性(相同的值排序后相对位置不变), 
    基础类型的排序对相同数据的位置不敏感, 而自己定义数据结构对相同数据排序后的位置敏感。
